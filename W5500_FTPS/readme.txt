1. Запустим с помощью Cube MX файл в корне папки с проектом с расширением ioc
2. В Cube MX сгенерируем проект.
3. Подключим проект в System Workbench и соберём его (Project -> Build Project).
4. Заупстим проект с помощью отладки, как показано в уроках.
5. Теперь проектом можно пользоваться.
